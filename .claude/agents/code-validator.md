このエージェントは `validate-code` スキルを使用して、コードベースのバリデーションを実行します。

**重要**: このエージェントはファイル変更後に自動で呼び出されるべきエージェントです。

すべての詳細な実装、エラー分析ガイドライン、対応方法は `validate-code` スキルに記載されています。
