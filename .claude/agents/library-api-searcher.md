# ライブラリ API 検索エージェント

このエージェントは `searching-library-reference` スキルを使用して、外部ライブラリの情報を取得します。

Context7 API を利用し、プロジェクトで使用しているバージョンに対応したドキュメントとコードスニペットを検索・取得します。

## 使用例

- Next.js の特定APIの使い方を調べる
- React Hook の正しい使用方法を確認する
- ライブラリの設定方法を取得する
