# チケット16: タスク一覧画面への新規作成リンク追加

## 概要

全タスク一覧画面（`/tasks`）に、タスク新規作成ページへのリンクを追加する。

## 背景

現在、タスク新規作成はプロジェクト詳細画面内のダイアログ形式でのみ提供されている。全タスク一覧画面（`/tasks`）からはタスクを新規作成する手段がなく、ユーザーがプロジェクト詳細画面へ遷移しなければならない。一覧画面から直接新規作成ページへ遷移できるようにすることで、操作性を向上させる。

## 受け入れ条件

- 全タスク一覧画面にタスク新規作成ページへのリンク（ボタン等）が表示される
- リンクをクリックすると新規作成ページ（`/tasks/new`）へ遷移する
- 新規作成ページにタスク作成フォームが表示される
- タスク作成後、全タスク一覧画面へリダイレクトされる

## 実装対象

- `apps/web/app/(private)/tasks/page.tsx` — 新規作成ページへのリンクを追加
- `apps/web/app/(private)/tasks/new/page.tsx` — 新規作成ページを新規作成
- `apps/web/features/task/register/` — 必要に応じて既存コンポーネントを流用
