# チケット 05: CSVファイルからのボランティア一括登録

## 概要

CSVファイルをアップロードしてボランティアをイベントに一括登録する機能を実装する。
エラーがある場合は全件拒否し、エラー内容を画面に表示する。

## 背景

大人数のボランティア名簿を手動で1件ずつ登録するのは非効率のため、CSV取り込み機能を提供する。
名簿の整合性が重要なため、部分取り込みは行わず、エラー時は全件拒否とする。

## 実装対象

### 画面

| URL | 内容 |
|---|---|
| `/events/[eventId]/csv-import` | CSV取り込み画面 |

### CSVフォーマット仕様

```
ヘッダー行あり
カラム: name,code,gender,participationDates
```

| カラム名 | 内容 | バリデーション |
|---|---|---|
| name | 氏名 | 必須、最大100文字 |
| code | 6桁数字ID | 必須、数字6桁 |
| gender | 性別 | 任意（空欄可） |
| participationDates | 参加予定日 | 必須、カンマ区切りで複数指定可（例: `2024-01-11,2024-01-12`）、イベント開催日に含まれること |

### バリデーション仕様

以下の条件のいずれかを満たす場合、全件拒否してエラー内容を表示する：

1. CSVのフォーマットが不正（ヘッダー不一致等）
2. 必須項目が空欄の行がある
3. `code` が数字6桁でない行がある
4. CSV内で `code` が重複している行がある
5. DB上で既存ボランティアと `code` が重複する行がある
6. `participationDates` にイベント開催日以外の日付が含まれる行がある

### エラー表示仕様

- エラーがある場合、行番号とエラー内容を一覧表示する
- 例：「3行目: IDは数字6桁で入力してください」
- エラー修正後に再アップロードしてもらう

### 取り込み成功時

- 取り込み件数を表示する
- ボランティア一覧ページ（`/events/[eventId]/volunteers`）にリダイレクトする

## features モジュール構成

```
apps/web/features/csv-import/
  components/
    csv-import-form.tsx     # ファイルアップロード・結果表示
    csv-error-list.tsx      # エラー一覧表示
  actions/
    import-volunteers-csv.ts
  lib/
    parse-csv.ts            # CSVパース・バリデーションロジック
```

## 実装手順

1. `features/csv-import/` モジュールを作成
2. `parse-csv.ts` にCSVパース・バリデーションロジックを実装
3. `import-volunteers-csv.ts` サーバーアクションを実装
   - CSVパース → バリデーション → 全件INSERT（トランザクション）
4. `csv-import-form.tsx`・`csv-error-list.tsx` を実装
5. `/events/[eventId]/csv-import` ページを実装
6. `pnpm -w validate:check` でエラーがないことを確認
7. `pnpm -w build` が通ることを確認
8. ブラウザで動作確認
   - 正常なCSVで取り込み成功
   - エラーのあるCSVで全件拒否・エラー表示
   - コード重複時にエラー表示
