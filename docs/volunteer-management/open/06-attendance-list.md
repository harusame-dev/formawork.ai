# チケット 06: 来場状況一覧（日付×ボランティアのマトリクス表示）

## 概要

イベント詳細画面（`/events/[eventId]`）にて、日付×ボランティアのマトリクス形式で来場状況を表示する機能を実装する。

## 背景

管理者がイベント当日や終了後に、各ボランティアの来場状況を一覧で把握できる必要がある。
マトリクス表示により、日付ごと・ボランティアごとの来場状況を視覚的に確認できる。

## 実装対象

### 画面

| URL | 内容 |
|---|---|
| `/events/[eventId]` | 来場状況一覧（イベント詳細画面のデフォルト表示） |

### マトリクス表示仕様

```
名前        1/11  1/12  1/13
───────────────────────────
根本将治     ◯    ✖️    ー
田中一郎     ◯    ◯    ◯
```

| 記号 | 意味 |
|---|---|
| ◯ | 来場済み（参加予定日に来場記録が1件以上あり） |
| ✖️ | 未来場（参加予定日だが来場記録なし） |
| ー | 非参加日（そもそも参加予定でない） |

### 表示仕様の詳細

- 列ヘッダー：イベントの全開催日（date[] から生成）
- 行：イベントに登録されているボランティア（氏名 + コードで識別）
- セル：当該ボランティアの当該日の来場状況
- ボランティア名はボランティア詳細ページへのリンク

### クエリ設計

- `events` → `volunteers` → `attendance_records` を JOIN
- ボランティアの `participationDates` と `attendance_records.recordedAt`（日付部分）を照合
- 来場記録は「その日に1件以上あるか」で判定（集計は DB か アプリ層で行う）

## features モジュール構成

```
apps/web/features/attendance/
  components/
    attendance-matrix.tsx       # マトリクス表示コンポーネント
    attendance-cell.tsx         # セル（◯/✖️/ー）
  queries/
    get-attendance-status.ts    # 来場状況クエリ
```

## 実装手順

1. `features/attendance/` モジュールを作成
2. `get-attendance-status.ts` を実装
   - イベントID を受け取り、ボランティアごと・日付ごとの来場状況を返す
3. `attendance-matrix.tsx`・`attendance-cell.tsx` を実装
4. `/events/[eventId]` ページに来場状況一覧を組み込む（チケット 03 のイベント詳細画面を拡張）
5. `pnpm -w validate:check` でエラーがないことを確認
6. `pnpm -w build` が通ることを確認
7. ブラウザで動作確認
   - 来場記録ありのボランティアに ◯ が表示される
   - 参加予定日だが来場記録なしのボランティアに ✖️ が表示される
   - 参加予定外の日付に ー が表示される
