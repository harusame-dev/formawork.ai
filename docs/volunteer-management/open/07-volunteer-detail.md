# チケット 07: ボランティア詳細（登録情報・来場記録タイムスタンプ）

## 概要

ボランティアの登録情報（氏名・コード・性別・参加予定日）と、来場記録の全タイムスタンプを表示する詳細画面を実装する。

## 背景

来場状況一覧（チケット 06）では日付単位の集計を表示するが、詳細な打刻履歴（1日に複数回打刻した場合のタイムスタンプ全件）を確認できる画面が必要。

## 実装対象

### 画面

| URL | 内容 |
|---|---|
| `/events/[eventId]/volunteers/[volunteerId]` | ボランティア詳細 |

### 表示内容

#### 登録情報セクション

| 項目 | 内容 |
|---|---|
| 氏名 | ボランティアの名前 |
| ID（コード） | 6桁数字コード |
| 性別 | 性別（未設定の場合は「-」） |
| 参加予定日 | イベント開催日の中から登録した日付一覧 |

#### 来場記録セクション

- `attendance_records.recordedAt` のタイムスタンプを全件・時系列順（新しい順）で表示
- 来場記録が0件の場合は「来場記録がありません」と表示
- 表示例：
  ```
  2024-01-11 09:32:15
  2024-01-11 13:05:42
  2024-01-12 10:01:08
  ```

#### ページアクション

- 編集ボタン → `/events/[eventId]/volunteers/[volunteerId]/edit`
- 削除ボタン（確認ダイアログあり）→ 削除後にボランティア一覧へリダイレクト
- 戻るリンク → `/events/[eventId]/volunteers`

## features モジュール構成

チケット 04 の `features/volunteer/` を拡張する：

```
apps/web/features/volunteer/
  components/
    volunteer-detail.tsx         # 詳細表示コンポーネント
    attendance-record-list.tsx   # 来場記録タイムスタンプ一覧
  queries/
    get-volunteer-with-records.ts  # ボランティア情報 + 来場記録クエリ
```

## 実装手順

1. `get-volunteer-with-records.ts` を実装
   - volunteerId を受け取り、ボランティア情報と来場記録（全件）を返す
2. `volunteer-detail.tsx`・`attendance-record-list.tsx` を実装
3. `/events/[eventId]/volunteers/[volunteerId]` ページを実装
4. `pnpm -w validate:check` でエラーがないことを確認
5. `pnpm -w build` が通ることを確認
6. ブラウザで動作確認
   - 登録情報が正しく表示される
   - 来場記録が全件タイムスタンプで表示される
   - 来場記録が0件の場合に空状態が表示される
