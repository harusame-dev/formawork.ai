# 命名規則

ファイルパスからそのファイルの種別を識別できるよう、以下の命名規則を遵守する。

## ファイル種別と命名パターン

テストのサイズ区分は [Google Testing Blog の Testing Size](https://testing.googleblog.com/2010/12/test-sizes.html) を基準とする。

| ファイル種別               | 命名パターン              |
| -------------------------- | ------------------------- |
| サーバーアクション         | `*.action.ts`             |
| サーバーコンポーネント     | `*.server.tsx`            |
| クライアントコンポーネント | `*.client.tsx`            |
| ユニバーサルコンポーネント | `*.universal.tsx`         |
| React hook                 | `use-*.hook.ts`           |
| ブラウザテスト             | `*.{small,medium}.browser.test.{ts,tsx}` |
| サーバーテスト             | `*.{small,medium}.server.test.ts`        |
| E2E テスト                 | `*.{medium,large}.e2e.test.ts`           |
